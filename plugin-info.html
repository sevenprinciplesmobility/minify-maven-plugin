Plugin Documentation

This report describes goals, parameters details, requirements and sample usage of this plugin.

Goals

Goals available for this plugin:

Goal Description
minify:help Display help information on minify-maven-plugin.
Call mvn minify:help -Ddetail=true -Dgoal=<goal-name> to display parameter details.
minify:minify Goal for combining and minifying CSS and JavaScript files.

System Requirements

The following specifies the minimum requirements to run this Maven plugin:

Maven 3.0
JDK 11

Usage

You should specify the version in your project's plugin configuration:

<project>
  ...
  <build>
    <!-- To define the plugin version in your parent POM -->
    <pluginManagement>
      <plugins>
        <plugin>
          <groupId>com.7p-group</groupId>
          <artifactId>minify-maven-plugin</artifactId>
          <version>1.7.12-SNAPSHOT</version>
        </plugin>
        ...
      </plugins>
    </pluginManagement>
    <!-- To use the plugin goals in your POM or parent POM -->
    <plugins>
      <plugin>
        <groupId>com.7p-group</groupId>
        <artifactId>minify-maven-plugin</artifactId>
      </plugin>
      ...
    </plugins>
  </build>
  ...
</project>

For more information, see "Guide to Configuring Plug-ins"
